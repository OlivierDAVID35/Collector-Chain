# projet-05-collector-chain

## Installation BACK

### Prérequis

- installer sqitch
- instaler postgreSQL
- créer une BDD "collector_chain"
- créer un fichier .env avec les informations de votre BDD "collector_chain"

1. Installer les differents modules

   ``` npm i ```

2. Importer la BDD

    ``` sqitch deploy ```
